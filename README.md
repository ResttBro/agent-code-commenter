# 🤖 Agent 智能代码修复与注释补全系统

**描述**：一个基于多Agent协作的智能代码修复与注释补全系统，专为理解和维护老旧后端代码库而设计。

## 🧠 系统架构
系统通过消息队列协同四个核心Agent工作，提升代码可维护性：
*   **扫描Agent**：扫描代码库，识别无注释的方法和潜在技术债务。
*   **分析与规划Agent**：利用长链推理深度理解业务逻辑，并生成具体的修复方案。
*   **执行Agent**：调用Cursor和Codex等AI工具，自动生成代码和注释。
*   **验证Agent**：自动运行测试，检查修复是否破坏现有功能，若失败则回滚并重新规划。

## 🚀 核心成果
*   已在小范围试点，每周降低约 **50%** 的代码审查时间。
*   近30天累计处理约 **3000万** Token，任务平均成功率 **94%**。
*   目标：全面迁移至 MiMo V2.5 模型以提升中文注释的生成质量。

## 📅 项目状态
*   **底层模型**：Claude 3.5 Sonnet (复杂任务)，DeepSeek V3 (日常代码生成)
*   **开发工具**：Cursor (主力)，Claude Code (长上下文)
